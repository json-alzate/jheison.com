Portafolio

Somiaz

Con el apoyo de Adsalutem, Somiaz es una app diseñada para ayudar a las personas a mejorar su descanso. Ofrece contenido paso a paso, como planes para dormir mejor, y permite registrar horas de sueño y actividad física conectando dispositivos como relojes inteligentes. Además, incluye un calendario para anotar eventos importantes y hábitos diarios relacionados con el bienestar.

Lullaai

(App destacada en Apple Store) Lullaai ayuda a los bebés a dormir mejor con inteligencia artificial que detecta despertares y reproduce sonidos calmantes. Incluye planes de sueño, un chat con sleep coaches, artículos útiles y un diario para registrar sueño, alimentación y cambios de pañal.

LullaLib

Librería que ha permitido crear aplicaciones móviles para el bienestar. Como Maternity+, Somiaz, Lullaai v2, Runners.
Apoyada por Google for Startups.

Chatia

Dashboard web para que las sleep coaches gestionen casos mediante un chat potenciado con inteligencia artificial. La plataforma permite hacer un seguimiento detallado de cada caso y generar reportes completos sobre las conversaciones.

Ionic Emoji

Librería de emojis para aplicaciones con Ionic.

UbankLib

Librería para aplicaciones móviles que capten ahorros interactivos para el sector bancario.

Renovación urbana

Esta aplicación se dividió en dos partes: un sitio web para capturar nuevos clientes potenciales y un panel para gestionar todo el proceso de alquiler o venta de una propiedad.

Whatsapp Chatbot

Un chatbot para gestionar citas médicas mediante la API de Twilio para WhatsApp. Este chatbot fue diseñado para ayudar a los pacientes a programar citas, recibir recordatorios y responder preguntas frecuentes. También incluye un panel para monitorear conversaciones y puntajes NPS.
